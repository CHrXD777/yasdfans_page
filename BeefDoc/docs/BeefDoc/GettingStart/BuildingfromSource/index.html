    从源代码构建 #

    构建概述 #

    在具有可用的二进制发行版的平台上（例如 Windows），无需从源代码构建 Beef。

    源代码可在 https://github.com/beefytech/Beef 获取。

    引导 #

    Beef 编译器的核心部分由 C++ 编写，而集成开发环境和命令行 BeefBuild 编译系统则由 Beef 本身编写。出于引导的目的，Beef 包含了一个最小的引导编译器，其唯一的工作就是执行初始的 BeefBuild 编译，然后再执行自身的 “正确 “构建。

    在 Windows 上构建 #

    要求 #

    • Visual Studio 2017 或更高版本的 Microsoft C++ 构建工具。您可以从 https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022 安装 Microsoft Visual C++ Build Tools 或整个 Visual Studio 套件。
    • Platform Toolset 141 (VS2017)
    • Windows SDK 10.0.17763.0
    • CMake 3.15 或更新版本
    • Python 3.6 或更新版本
    • Git 命令行工具

    构建步骤 #

    • 执行 bin/build.bat

    注意，这将首先下载并编译 LLVM，这需要一些时间。编译结果将保存在 IDE/dist 中。

    在 Linux 和 macOS 上构建 #

    要求 #

    • CMake 3.15 或更新版本
    • Python 2.7
    • Git

    构建步骤 #

    • 使用 bin/build.sh 构建 Beef

    这将构建 LLVM 等依赖项，这可能需要相当长的时间。

    编译结果将保存在 IDE/dist 中。

    请注意，这些平台均支持 BeefBuild 等 CLI 工具，但 IDE 目前仅适用于 Windows。
