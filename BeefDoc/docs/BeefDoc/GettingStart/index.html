    入门 #

    欢迎使用 Beef #

    Beef 主要基于集成开发环境，但也支持命令行构建。第一步是安装 Beef 或从源代码构建。目前，二进制文件仅适用于 Windows，Beef IDE 也仅适用于 Windows。

    支持的平台 #

    二进制文件可用于 Windows，并支持在 Windows、Linux 和 macOS 上从源代码构建。针对 Android 和 iOS 等目标的交叉编译正在开发中。

    创建基于 IDE 的“Hello World” #

    • 运行 Beef IDE。
    • 通过选择 File/New/Project 创建一个名为 “Hello” 的新控制台项目。
    • 右键单击新项目，选择 “New Class…"，并输入 “Program” 作为名称。
    • 在新创建的文件中输入以下文本。
    using System;

namespace Hello
{
    class Program
    {
        static void Main()
        {
            Console.WriteLine("Hello, world!");
        }
    }
}
    • 按下 F5 编译并执行

    创建基于命令行的“Hello World” #

    • 在某个地方创建一个名为 “Hello” 的新目录。
    • 使用命令行进入该目录。
    • 运行 “BeefBuild -new”，在该目录中初始化一个新的工作区和项目。
    • 创建一个 “src/Program.bf” 文本文件，并将上述方框中的内容粘贴进去。
    • 运行 “BeefBuild -run” 来编译并执行。
